À propos de mes compétences !

Durant tous ses projets réalisés à l'IUT ou personnellement, j'ai pu acquérir de nombreuses compétences, dont certaines avec une grande maitrise. Étant plus passionné par le web, j'ai pu me spécialiser dans certains frameworks et langages de programmation web.

Framework Web

Angular

Angular est un framework open-source développé par Google, utilisé pour créer des applications web dynamiques et interactives. Il repose sur TypeScript et permet de construire des applications côté client de manière modulaire, avec une architecture basée sur des composants. Angular inclut des outils puissants pour la gestion des formulaires, des routages, des requêtes HTTP

VueJs

Vue.js est un framework JavaScript open-source utilisé pour créer des interfaces utilisateur et des applications web dynamiques. Il est léger, flexible et facile à intégrer dans des projets existants. Vue adopte une architecture basée sur des composants, permettant une gestion claire et réutilisable du code. Il propose également des fonctionnalités puissantes comme la gestion des états avec Vuex et le routage avec Vue Router, tout en étant apprécié pour sa simplicité et sa courbe d'apprentissage progressive.

NodeJs

Node.js est un environnement d'exécution JavaScript côté serveur, construit sur le moteur V8 de Google Chrome. Il permet d'exécuter du code JavaScript en dehors d'un navigateur, ce qui est particulièrement utile pour créer des applications web rapides et scalables. Node.js utilise un modèle non-bloquant et orienté vers les événements, ce qui le rend très performant pour gérer de nombreuses connexions simultanées.

Express

Express est un framework web minimaliste et flexible pour Node.js. Il simplifie la création d'applications web et d'API en fournissant des outils puissants pour gérer les routes, les requêtes HTTP, les middlewares, et les vues. Express permet de développer rapidement des applications serveur, en offrant une structure légère mais extensible, idéale pour des projets allant des petites applications aux systèmes plus complexes.

Langages de programmation

HTML/CSS

HTML (HyperText Markup Language) est le langage de balisage standard utilisé pour structurer le contenu des pages web. Il définit les éléments de la page, tels que les titres, les paragraphes, les images, les liens, et les formulaires, à l'aide de balises.

Python

Python est un langage de programmation polyvalent, facile à apprendre et à lire, utilisé pour le développement d'applications web, l'analyse de données, l'intelligence artificielle, l'automatisation, et bien plus. Sa syntaxe simple et sa grande communauté en font un choix populaire pour les débutants et les professionnels. Python dispose de nombreuses bibliothèques et frameworks puissants, comme Django, Flask, NumPy, et Pandas, ce qui le rend adapté à une vaste gamme d'applications.

Typescript

TypeScript est un sur-ensemble de JavaScript qui ajoute des fonctionnalités de typage statique. Il permet aux développeurs de définir des types de données (comme les chaînes de caractères, les nombres, ou les objets), ce qui aide à détecter les erreurs de manière précoce lors du développement.

C++

c++ est un langage de programmation puissant, orienté objet et de bas niveau, largement utilisé pour le développement de logiciels système, d'applications à haute performance, de jeux vidéo, et d'outils embarqués. Il offre un contrôle précis de la gestion de la mémoire et des ressources, ce qui permet de créer des programmes très rapides et efficaces.

Java

Java est un langage de programmation orienté objet, utilisé pour créer des applications web, des applications mobiles (notamment sur Android), des logiciels d'entreprise, et des systèmes embarqués. Il est conçu pour être portable, ce qui signifie qu'un programme Java peut être exécuté sur n'importe quelle plateforme qui dispose d'une machine virtuelle Java (JVM)

Outils

Visual Studio

Visual Studio Code (VS Code) est un éditeur de code source léger et open-source développé par Microsoft. Il est conçu pour être rapide, personnalisable et extensible, avec un large éventail d'extensions disponibles pour différents langages de programmation (comme JavaScript, Python, C++, etc.) et frameworks.

Figma

Figma est un outil de design collaboratif basé sur le cloud, utilisé pour créer des interfaces utilisateurs (UI), des prototypes et des maquettes. Il permet aux designers de travailler ensemble en temps réel, de partager des fichiers et d'itérer facilement sur des projets.

Github

GitHub est une plateforme de développement logiciel et de gestion de code source basée sur Git, qui permet aux développeurs de stocker, de partager et de collaborer sur des projets de programmation. Il offre des fonctionnalités telles que le suivi des versions, la gestion des branches, les pull requests et les problèmes (issues), facilitant ainsi la collaboration entre développeurs.

Docker

Docker est une plateforme open-source qui permet de développer, expédier et exécuter des applications dans des conteneurs. Les conteneurs sont des environnements légers, isolés et portables qui contiennent tout le nécessaire pour faire fonctionner une application (code, bibliothèques, dépendances, etc.).

Gestion de projet

Les applications de gestion de projet sont des outils conçus pour aider les équipes à organiser, suivre et gérer l'avancement de leurs projets. Elles permettent de planifier les tâches, de définir des objectifs, d'affecter des responsabilités et de suivre les délais.
